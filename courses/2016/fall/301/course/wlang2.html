3.16 Language Analysis the Second

The suggested length is 10 pages.

Repeat the Language Analysis the First assignment, although this time include more detail.

You should plan on choosing a different language. However, it is possible that I may suggest you revise your first paper and improve it. This revision is reflected in the greater length. If you are offerred a resubmission, be sure to improve all parts of the paper. (For example, I suggest that you improve A, B, and C, then you improve A and the paper would be 650 lines if you deleted B and C, you should not delete B and C, but should improve everything and arrive at more than 650 lines.)

If you must choose a different language, but choose one that is basically the same (for example, JavaScript to ActionScript), you’ll be disqualified.
